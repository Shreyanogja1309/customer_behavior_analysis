Data Analytics Project
1️. Overview

This project demonstrates an end-to-end Data Analytics workflow — from data loading and cleaning to SQL analysis and dashboard visualization.

The objective of this project is to:

Load and analyze raw data using Python

Perform Exploratory Data Analysis (EDA)

Clean and preprocess the dataset

Run advanced SQL queries using relational databases

Build an interactive Power BI dashboard

Generate a business report

Create a professional presentation using Gamma

This project showcases skills required for a Data Analyst role, including data manipulation, database querying, visualization, and business storytelling.
